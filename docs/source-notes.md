# Source Notes

This starter kit is built around public guidance and platform requirements.
It does not interpret the law for your specific situation.

## Common Disclosure Themes

Most privacy notices should help a user understand:

- who operates the service
- what data is collected
- why the data is used
- who receives the data
- how long data is kept
- how users can contact the operator
- how users can exercise privacy or deletion choices
- whether data is transferred internationally
- whether automated or AI processing is involved

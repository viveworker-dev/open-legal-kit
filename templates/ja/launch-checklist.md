## 基本項目

- [ ] すべてのプレースホルダーを実際のアプリ情報に置き換えた。
- [ ] 運営者名が、アプリストア掲載情報、Webサイトのフッター、決済アカウントと一致している。
- [ ] Webサイトのフッターにプライバシーポリシーと利用規約へのリンクを追加した。
- [ ] アプリ内の自然な場所にプライバシーポリシーとアカウント削除リンクを追加した。
- [ ] 連絡先メールアドレスが機能することを確認した。
- [ ] 保存期間が実際のデータベースおよびログ保存期間と一致している。
- [ ] 第三者処理事業者が実際に使用しているサービスと一致している。

## データフロー確認

- [ ] メールアドレスおよび連絡先データが開示されている。
- [ ] アカウントデータが開示されている。
- [ ] 分析ツールが開示されている。
- [ ] Cookieまたはローカルストレージが開示されている。
- [ ] AIプロバイダーおよびプロンプト/出力の扱いが開示されている。
- [ ] アップロードファイルおよびユーザーコンテンツが開示されている。
- [ ] 決済処理事業者および請求データが開示されている。
- [ ] サポートメッセージおよび不正利用/セキュリティログが開示されている。

## 用途別確認

- [ ] AI wrapper アプリでは、第三者AIプロバイダーおよびプロンプト/出力の扱いが開示されている。
- [ ] ファイルアップロードアプリでは、アップロード保存期間、削除、センシティブアップロードのリスクが開示されている。
- [ ] 有料SaaSでは、サブスクリプション、解約、返金、プラン変更、利用上限が開示されている。
- [ ] モバイルアプリでは、アプリストアリンク、端末権限、プッシュ通知、クラッシュレポートが開示されている。
- [ ] アカウントなしツールでは、アカウント識別子なしで削除/開示請求に対応する方法が説明されている。

## プロダクト確認

- [ ] 法務レビューなしに子ども向けサービスになっていない。
- [ ] 法務レビューなしに健康、金融、雇用、生体情報、その他センシティブデータを処理していない。
- [ ] アプリストアのプライバシーラベルがプライバシーポリシーと一致している。
- [ ] ランディングページ上の主張が、実際のアプリ機能と一致している。
- [ ] 返金およびサブスクリプション条件が決済フローと一致している。
- [ ] プラットフォームで必要な場合、アプリ内外からアカウント削除が可能である。
